## About the Collaboration

This project was designed and directed by Abhinav Sabbarwal. The core idea, structure, and feature set came from his own engineering growth journey — the realization that SDE interview prep is a messy, time-bound curriculum when it should be adaptive, personalized, and pace-driven by the learner.

Claude (via Claude Code) served as an engineering collaborator:
- Drafted initial SKILL.md structure from Abhinav's system prompt
- Implemented hot-topics.json schema
- Built template files (onboarding, dashboard, weekly review)
- Refactored based on iterative feedback
- Ensured portable paths for community use

The direction, vision, and final decisions remain Abhinav's. The code execution was augmented with Claude.
